# Block V7 (Monorepo)

This is the merged V7 monorepo scaffold.

- `apps/manager-web`: Cloudflare Pages manager UI (React+TS)
- `apps/api`: Railway API + Worker (Express + Supabase Admin + Twilio + Stripe)
- `apps/rep-mobile/_import_raw`: imported rep app (needs adaptation to V7)
- `apps/labor-mobile`: new Labor RN app (Expo) with Stripe PaymentSheet flow
- `packages/shared`: shared types + event taxonomy
- `supabase`: migrations + seed template
- `docs`: setup + deployment

See `docs/DEPLOYMENT.md`.
